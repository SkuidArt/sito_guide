Selettori di Tipo

I selettori di tipo mirano a un elemento HTML specifico. Ad esempio, per stilizzare tutti gli elementi h1, puoi utilizzare il seguente codice CSS:

              

                                
                h1 {
                  /* I tuoi stili qui */
                }
Qui, il selettore h1 mira a tutti gli elementi h1 nel documento HTML.

Selettori di Classe

I selettori di classe mirano agli elementi con un valore specifico per il loro attributo di classe. Considera l'esempio seguente, dove il selettore di classe stila gli elementi con la classe "box":

              

                
                  .box {
                    /* I tuoi stili qui */
                  }
In HTML, gli elementi con la classe "box" verrebbero selezionati e stilizzati di conseguenza.

Selettori di Id

I selettori di ID mirano agli elementi con un valore specifico per il loro attributo di ID. Il seguente codice CSS dimostra come stilizzare un elemento con l'ID "unique":

              

                
                  #unique {
                    /* I tuoi stili qui */
                  }
In HTML, l'elemento con l'ID "unique" verrebbe stilizzato in modo univoco.

Pseudo-classi e Pseudo-elementi

Questo gruppo di selettori include pseudo-classi, che stiliano determinati stati di un elemento. Ad esempio, la pseudo-classe :hover seleziona un elemento solo quando viene sovrapposto dal puntatore del mouse:

              

                
                  a:hover {
                    /* I tuoi stili qui */
                  }

Include anche pseudo-elementi, che selezionano una parte specifica di un elemento anziché l'elemento stesso. Ad esempio, ::first-line seleziona sempre la prima riga di testo all'interno di un elemento p, comportandosi come se un span fosse avvolto attorno alla prima riga formattata e poi selezionato.

              

                
                  p::first-line {
                    /* I tuoi stili qui */
                  }

Combinatori

I combinatori consentono di unire più selettori per mirare a elementi specifici nei documenti. Nell'esempio seguente, si selezionano i paragrafi che sono figli diretti degli elementi article utilizzando il combinatore di figlio (>):

              

                
                  article > p {
                    /* I tuoi stili qui */
                  }
                  //oppure
                  article p {
                    /* I tuoi stili qui */
                  }

Selettori di Attributo

Questi selettori consentono la selezione di un elemento basandosi sulla presenza di un attributo o su diverse corrispondenze contro il valore dell'attributo.

Selettore Esempio Descrizione
[attr] a[title] Seleziona gli elementi con un attributo "attr" (il cui nome è il valore tra parentesi quadre).
[attr=value] a[href="https://example.com"] Seleziona gli elementi con un attributo "attr" il cui valore è esattamente "value".
[attr~=value] p[class~="special"] Seleziona gli elementi con un attributo "attr" il cui valore contiene "value" nella sua lista di valori (separati da spazi).
[attr|=value] div[lang|="zh"] Seleziona gli elementi con un attributo "attr" il cui valore inizia con "value" immediatamente seguito da un trattino.

Selettori di Corrispondenza di Sottostringhe

Questi selettori consentono una corrispondenza più avanzata delle sottostringhe all'interno del valore del tuo attributo. Ad esempio, se avessi classi come box-warning e box-error e volessi selezionare tutto ciò che inizia con la stringa "box-", potresti usare [class^="box-"] per selezionarle entrambe (o [class|="box"] come descritto nella sezione precedente).

Selettore Esempio Descrizione
[attr^=value] li[class^="box-"] Seleziona gli elementi con un attributo "attr" il cui valore inizia con "value".
[attr$=value] li[class$="-box"] Seleziona gli elementi con un attributo "attr" il cui valore termina con "value".
[attr*=value] li[class*="box"] Seleziona gli elementi con un attributo "attr" il cui valore contiene "value" in qualsiasi parte della stringa.

Pseudo Classi

Una pseudo-classe è un selettore che seleziona gli elementi che si trovano in uno stato specifico, ad esempio, sono il primo elemento del loro tipo o stanno per essere evidenziati dal puntatore del mouse.

Tendono ad agire come se avessi applicato una classe a una parte del tuo documento, spesso aiutandoti a ridurre l'eccesso di classi nel tuo markup e fornendoti un codice più flessibile e manutenibile.

Le pseudo-classi sono parole chiave che iniziano con due punti. Ad esempio, :hover è una pseudo-classe.

Esempio semplice di pseudo-classe

Consideriamo un esempio semplice. Se volessimo rendere il primo paragrafo in un articolo più grande e in grassetto, potremmo aggiungere una classe a quel paragrafo e quindi aggiungere CSS a quella classe, come mostrato nel primo esempio qui sotto:

              

                
                  .first {
                    font-size: 120%;
                    font-weight: bold;
                  }

                  <article>
                  <p class="first">Veggies es bonus vobis, proinde vos postulo essum magis kohlrabi welsh onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.</p>

                  <p>Gumbo beet greens corn soko endive gumbo gourd. Parsley shallot courgette tatsoi pea sprouts fava bean collard
                  greens dandelion okra wakame tomato. Dandelion cucumber     earthnut pea peanut soko zucchini.</p>
                  </article>

Tuttavia, questo potrebbe essere fastidioso da mantenere: e se venisse aggiunto un nuovo paragrafo all'inizio del documento? Dovremmo spostare la classe al nuovo paragrafo.

Invece di aggiungere la classe, potremmo utilizzare il selettore :first-child, una pseudo-classe che mirerà sempre al primo elemento figlio nell'articolo, e non dovremo più modificare l'HTML (questo potrebbe non essere sempre possibile comunque, forse a causa della generazione da parte di un CMS).

              

                
                  article p:first-child {
                    font-size: 120%;
                    font-weight: bold;
                  }

                  <article>
                  <p>Veggies es bonus vobis, proinde vos postulo essum magis kohlrabi welsh onion daikon amaranth tatsoi tomatillo
                  melon azuki bean garlic.</p>

                  <p>Gumbo beet greens corn soko endive gumbo gourd. Parsley shallot courgette tatsoi pea sprouts fava bean collard
                  greens dandelion okra wakame tomato. Dandelion cucumber earthnut pea peanut soko zucchini.</p>
                  </article>

Tutte le pseudo-classi si comportano in questo stesso modo. Mirano a una parte del tuo documento che si trova in uno stato specifico, comportandosi come se avessi aggiunto una classe nel tuo HTML. Dai un'occhiata ad alcuni altri esempi su MDN:

  1. :last-child
  2. :only-child
  3. :invalid
Nota:
È valido scrivere pseudo-classi ed elementi senza alcun selettore di elemento che li precede. Nell'esempio sopra, potresti scrivere :first-child e la regola si applicherebbe a qualsiasi elemento che è il primo figlio di un elemento article, non solo a un paragrafo primo figlio. :first-child è equivalente a *:first-child. Tuttavia, di solito si desidera maggiore controllo, quindi è necessario essere più specifici.

Colore del Testo

La proprietà color viene utilizzata per impostare il colore del testo. Il colore può essere specificato mediante:

  • un nome di colore, come "red" (rosso),
  • un valore esadecimale HEX, come "#ff0000",
  • un valore RGB, come "rgb(255,0,0)".

Il colore predefinito del testo per una pagina è definito nel selettore body.

                

                                  
                    body {
                      color: blue;
                    }
                    
                    h1 {
                      color: green;
                    }
body { color: blue; }: Il selettore body si riferisce all'elemento body del documento HTML, che rappresenta il corpo della pagina. La proprietà color viene utilizzata per impostare il colore del testo all'interno dell'elemento body. Nel nostro caso, il colore del testo è impostato su blu (blue). h1 { color: green; }: Il selettore h1 si riferisce a tutti gli elementi di intestazione di primo livello h1 nel documento HTML. La proprietà color viene utilizzata per impostare il colore del testo all'interno degli elementi h1. Nel nostro caso, il colore del testo per gli elementi h1 è impostato su verde (green). In generale, questi stili verranno applicati a tutti gli elementi body e h1 nella tua pagina HTML, a meno che non vengano sovrascritti da stili più specifici o da altri fogli di stile CSS. Questi stili determinano che il testo nel corpo della pagina sarà blu, mentre il testo negli elementi di intestazione di primo livello (h1) sarà verde.

Allineamento del Testo

La proprietà text-align viene utilizzata per impostare l'allineamento orizzontale di un testo.

Un testo può essere allineato a sinistra o a destra, centrato o giustificato.

Nell'esempio seguente viene mostrato un testo centrato e testi allineati a sinistra e a destra (l'allineamento a sinistra è predefinito se la direzione del testo è da sinistra a destra, e l'allineamento a destra è predefinito se la direzione del testo è da destra a sinistra):

                

                                  
                    h1 {
                      text-align: center;
                    }
                    
                    h2 {
                      text-align: left;
                    }
                    
                    h3 {
                      text-align: right;
                    }

Quando la proprietà text-align è impostata su "justify", ogni riga viene estesa in modo che ogni riga abbia larghezza uguale, e i margini sinistro e destro sono dritti (come nei giornali e nelle riviste):

NB: non è consigliabile usare questa proprietà per giustificare il testo in quanto non è di facile lettura. 

                

                                  
                    div {
                      text-align: justify;
                    }

Testo Allineato all'Ultimo

La proprietà text-align-last specifica come allineare l'ultima riga di un testo. Esempio: Allinea l'ultima riga di testo in tre elementi p:

                

                                  
                    p.a {
                      text-align-last: right;
                    }
                    
                    p.b {
                      text-align-last: center;
                    }
                    
                    p.c {
                      text-align-last: justify;
                    }

Nell'esempio sopra, i paragrafi con le classi a, b, e c hanno differenti allineamenti dell'ultima riga del testo. Il paragrafo con la classe a allinea l'ultima riga a destra, il paragrafo con la classe b la allinea al centro, mentre il paragrafo con la classe c la giustifica.

Direzione del Testo

Le proprietà direction e unicode-bidi possono essere utilizzate per cambiare la direzione del testo di un elemento:

                

                                  
                    p {
                      direction: rtl;
                      unicode-bidi: bidi-override;
                    }

direction: rtl;: Questo imposta la direzione del testo a "destra verso sinistra" (Right-to-Left). Ciò significa che il testo all'interno dell'elemento, come un paragrafo p, sarà scritto e visualizzato da destra verso sinistra.

unicode-bidi: bidi-override;: Questo specifica la direzione del testo e il comportamento del bidirezionale (bidi) del testo. Con il valore "bidi-override", si indica che il testo deve essere sovrascritto bidirezionalmente in base alle direzioni specificate. Questo è spesso utilizzato in combinazione con direction: rtl; per gestire correttamente il testo in lingue che richiedono la scrittura da destra a sinistra.

In sostanza, questo blocco di stile CSS modifica la direzione del testo del paragrafo

, facendo sì che il testo venga visualizzato da destra verso sinistra e utilizzando le regole di scrittura bidirezionale specificate.

Allineamento Verticale

La proprietà vertical-align imposta l'allineamento verticale di un elemento.

Esempio - Imposta l'allineamento verticale di un'immagine all'interno di un testo:

                

                                  
                    img.a {
                      vertical-align: baseline;
                    }
                    
                    img.b {
                      vertical-align: text-top;
                    }
                    
                    img.c {
                      vertical-align: text-bottom;
                    }
                    
                    img.d {
                      vertical-align: sub;
                    }
                    
                    img.e {
                      vertical-align: super;
                    }

img.a { vertical-align: baseline; }: Allinea l'immagine rispetto alla "baseline" del testo circostante. La "baseline" è la linea di base sulla quale si appoggiano i caratteri di testo.

img.b { vertical-align: text-top; }: Allinea la parte superiore dell'immagine con la parte superiore del contenuto testuale più alto nell'elemento padre.

img.c { vertical-align: text-bottom; }: Allinea la parte inferiore dell'immagine con la parte inferiore del contenuto testuale più basso nell'elemento padre.

img.d { vertical-align: sub; }: Allinea l'immagine come se fosse un indice (un po' più in basso rispetto alla linea di base).

img.e { vertical-align: super; }: Allinea l'immagine come se fosse un apice (un po' più in alto rispetto alla linea di base).

Queste regole vengono utilizzate per posizionare l'immagine verticalmente all'interno del testo in base alle diverse opzioni di allineamento specificate dalla proprietà vertical-align. La scelta dipende spesso dalle esigenze di layout specifiche dell'elemento all'interno del contesto della pagina web.

Aggiungere una Linea di Decorazione al Testo

La proprietà text-decoration-line viene utilizzata per aggiungere una linea di decorazione al testo.

Suggerimento: Puoi combinare più di un valore, come overline e underline, per visualizzare linee sia sopra che sotto un testo.

                

                                  
                    h1 {
                      text-decoration-line: overline;
                    }
                    
                    h2 {
                      text-decoration-line: line-through;
                    }
                    
                    h3 {
                      text-decoration-line: underline;
                    }
                    
                    p {
                      text-decoration-line: overline underline;
                    }

Nota: Non è consigliato sottolineare il testo che non è un link, poiché questo spesso può confondere il lettore.

  • h1: Aggiunge una linea di decorazione sopra l'elemento h1.
  • h2: Aggiunge una linea di decorazione attraverso l'elemento h2.
  • h3: Sottolinea l'elemento h3.
  • p: Combina linee di decorazione sopra e sotto il testo all'interno degli elementi p.

Le linee di decorazione del testo sono spesso utilizzate in modo mirato per evidenziare determinate parti del contenuto.

  • h3: Imposta una sottolineatura dell'elemento h3 con il colore verde (green).
  • p: Combina una linea di decorazione sopra e sotto il testo all'interno degli elementi p con il colore viola (purple).

Questi esempi mostrano come è possibile personalizzare il colore delle linee di decorazione per evidenziare e stilizzare il testo in modo più specifico.

  • p.ex1: Imposta una sottolineatura per l'elemento p con uno stile tratteggiato (dashed).
  • p.ex2: Imposta una sottolineatura per l'elemento p con uno stile ondulato (wavy).

Specificare lo Spessore della Linea di Decorazione

La proprietà text-decoration-thickness viene utilizzata per impostare lo spessore della linea di decorazione.

                

                                  
                    h1 {
                      text-decoration-line: underline;
                      text-decoration-thickness: auto;
                    }
                    
                    h2 {
                      text-decoration-line: underline;
                      text-decoration-thickness: 5px;
                    }
                    
                    h3 {
                      text-decoration-line: underline;
                      text-decoration-thickness: 25%;
                    }
                    
                    p {
                      text-decoration-line: underline;
                      text-decoration-color: red;
                      text-decoration-style: double;
                      text-decoration-thickness: 5px;
                    }
  • h1: Imposta una sottolineatura per l'elemento h1 con uno spessore automatico (auto), che di solito dipende dal browser e dal contesto.
  • h2: Imposta una sottolineatura per l'elemento h2 con uno spessore di 5 pixel (5px).
  • h3: Imposta una sottolineatura per l'elemento h3 con uno spessore del 25% (25%), relativo alla larghezza del carattere.
  • p: Imposta una doppia sottolineatura rossa per l'elemento p con uno spessore di 5 pixel (5px).

La proprietà text-decoration-thickness permette di regolare lo spessore delle linee di decorazione, offrendo maggiore controllo sulle dimensioni visive della decorazione del testo.

La proprietà text-decoration

La proprietà text-decoration è una proprietà abbreviata per:

  • text-decoration-line (obbligatorio)
  • text-decoration-color (opzionale)
  • text-decoration-style (opzionale)
  • text-decoration-thickness (opzionale)
                

                                  
                    h1 {
                      text-decoration: underline;
                    }
                    
                    h2 {
                      text-decoration: underline red;
                    }
                    
                    h3 {
                      text-decoration: underline red double;
                    }
                    
                    p {
                      text-decoration: underline red double 5px;
                    }

  • h1: Imposta una sottolineatura (underline) per l'elemento h1
  • h2: Imposta una sottolineatura (underline) e il colore rosso (red) per l'elemento h2.
  • h3: Imposta una sottolineatura (underline), il colore rosso (red), e uno stile di linea doppia (double) per l'elemento h3
  • p: Imposta una sottolineatura (underline), il colore rosso (red), uno stile di linea doppia (double), e uno spessore di 5 pixel (5px) per l'elemento p.
Un piccolo suggerimento: Tutti i collegamenti in HTML sono sottolineati per impostazione predefinita. A volte si vuole che i collegamenti siano stilizzati senza sottolineatura. La regola text-decoration: none; viene utilizzata per rimuovere la sottolineatura dai collegamenti, come mostrato nell'esempio seguente:

                  

                                    
                      a {
                        text-decoration: none;
                      }

Questa regola viene spesso utilizzata per personalizzare lo stile dei collegamenti, ad esempio per farli apparire come pulsanti senza sottolineatura.

Trasformazione del Testo

La proprietà text-transform viene utilizzata per specificare lettere maiuscole e minuscole in un testo.

Può essere utilizzata per trasformare tutto in lettere maiuscole o minuscole, oppure per capitalizzare la prima lettera di ogni parola:

                  

                                    
                      p.uppercase {
                        text-transform: uppercase;
                      }
                      
                      p.lowercase {
                        text-transform: lowercase;
                      }
                      
                      p.capitalize {
                        text-transform: capitalize;
                      }
  • p.uppercase: Imposta il testo all'interno degli elementi p in lettere maiuscole

  • p.capitalize: Capitalizza la prima lettera di ogni parola nel testo all'interno degli elementi p.

Questi sono utili quando si desidera controllare il modo in cui il testo è visualizzato, ad esempio quando si desidera applicare uno stile specifico come tutto maiuscolo, tutto minuscolo o la capitalizzazione delle prime lettere delle parole.

Spazio Bianco

La proprietà white-space specifica come viene gestito lo spazio bianco all'interno di un elemento.

Questo esempio illustra come disabilitare il testo a capo all'interno di un elemento:

                  

                                    
                      p {
                        white-space: nowrap;
                      }

La regola white-space: nowrap; applicata agli elementi p impedisce che il testo all'interno del paragrafo non andrà a capo automaticamente alla fine della linea, ma continuerà su una singola linea, creando una barra di scorrimento orizzontale se il testo è più lungo del contenitore.

Questo può essere utile in situazioni in cui si desidera che il testo rimanga su una singola linea, ad esempio in una barra di navigazione orizzontale. Puoi regolare la gestione dello spazio bianco in base alle tue esigenze utilizzando altre opzioni come normal, pre, pre-line, o pre-wrap. Queste opzioni determinano come vengono trattati gli spazi bianchi, i ritorni a capo e il riavvolgimento del testo all'interno dell'elemento.

Effetto ombra del testo!

                      }
                      h2 {

                      }

                        text-shadow: 2px 2px 5px red;
                      }  
                      h4 {
                        color: black;

                      }
                      h5 {
                        text-shadow: 0 0 3px #ff0000;
                      } 
                      h6 {
                        text-shadow: 0 0 3px #ff0000, 0 0 5px #0000ff;

Gli esempi aggiuntivi mostrano come variare i colori e le dimensioni dell'ombra per ottenere effetti visivamente accattivanti. Questa proprietà può essere utilizzata per migliorare la leggibilità o per aggiungere stile al testo all'interno della tua pagina.
